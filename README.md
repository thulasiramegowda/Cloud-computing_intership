# ☁️ Cloud Computing Internship Projects

This repository contains my hands-on cloud computing internship projects completed as part of my learning journey with Decode Labs and AWS.

The goal of this repository is to document not only the final implementation but also the learning process, challenges faced, solutions explored, references used, and project screenshots.

## 🚀 Projects Included

### 📁 Project 1 - Static Website Hosting using AWS S3
- Hosted my personal portfolio using AWS S3 Static Website Hosting.
- Explored bucket policies, permissions, and public access settings.
- Learned how cloud object storage can be used for web hosting.

### 📁 Project 2 - EC2 Server Provisioning
- Launched and configured an AWS EC2 instance.
- Explored security groups, networking, and SSH access.
- Learned the basics of cloud virtual machines and server management.

### 📁 Project 3 - Managed Database using AWS RDS
- Created a managed MySQL database using AWS RDS.
- Created tables and inserted dummy records.
- Connected to the database through AWS CloudShell.
- Explored database connectivity and cloud data persistence.
