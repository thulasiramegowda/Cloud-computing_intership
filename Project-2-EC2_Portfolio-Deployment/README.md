# 🚀 Project 2 - AWS EC2 Server Provisioning

## ☁️ Overview
This project focused on understanding cloud virtual machines using AWS EC2 and learning how remote Linux servers can be accessed and managed securely from a Windows machine using SSH.

## 🔍 What I Explored
✔️ AWS EC2 Instances  
✔️ SSH and Remote Access  
✔️ Linux Commands and Navigation  
✔️ Security Groups and Network Rules  
✔️ Key Pairs and Authentication  
✔️ Git and GitHub on Remote Servers  
✔️ Portfolio Deployment on Cloud Infrastructure

## 🎯 Outcome
Successfully launched an EC2 instance and connected to a remote Linux server using SSH.

As an initial step, I first tested the deployment workflow using a simple `app.txt` file to understand how remote servers handle files and applications.

After gaining confidence with the setup, I took it a step further by deploying my own portfolio project on the EC2 instance to better understand real-world cloud deployment workflows.

## 🌱 Key Takeaway
One of the biggest learnings from this project was understanding that cloud servers run on remote infrastructure and can be securely managed and used to host applications from anywhere in the world.
