# 🗄️ Project 3 - AWS RDS Database and EC2 Integration

## ☁️ Overview
This project focused on understanding managed cloud databases using AWS RDS and exploring how compute resources and databases communicate in real-world cloud environments.

## 🔍 What I Explored
✔️ AWS RDS MySQL Database  
✔️ AWS EC2 Instances  
✔️ AWS CloudShell  
✔️ SQL Queries and Database Operations  
✔️ Security Groups and Connectivity  
✔️ Cloud Resource Management

## 🎯 Outcome
Successfully created and configured an AWS RDS MySQL database, created tables, inserted dummy records, and verified database operations using SQL queries.

As an additional learning step, I also launched an EC2 instance and explored how cloud applications communicate with managed databases in real-world architectures.

## 🌱 Key Takeaway
One of the biggest learnings from this project was understanding the separation between compute services (EC2) and database services (RDS), and how they work together to power cloud applications.
